#  Sistema DistriSys
-----
Sistema python de gerenciamneto operacional e logístico de distribuidora que realiza vendas de grande volume para comércios.

As tecnologias usadas são:
  - Python para toda a parte lógica 
  - Streamlit: Biblioteca Python para a criação da interface gráfica
  - Pandas: Biblioteca Python para análise de dados dos sistema
